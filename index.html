cnode社区移动端--Vue版

利用conde社区网站提供的API,使用vue 2.0框架,使用vue-cli搭建项目， 技术栈：vue 2.0 + vue-router + webpack + ES6 + sass。

青蛙过河游戏

使用HTML5的Canvas绘制所有的游戏元素，使用JavaScript的面向对象对玩家类和虫子类进行编写， 使用requestAnimationFrame来调用游戏的循环函数，使应用的运行速度尽可能达到 60帧每秒

网站性能优化项目

项目主要针对加载、动画、响应过程进行了优化 使用gulp工具压缩html、css、js等文件， 使用pageSpeed Insights 测试达到90分， 使用Chrome的performance检查pizza.html的性能问题， 优化后pizza.html滚动时保持60fps 的帧速

街区地图

使用 gulp 工具构建该项目，使用 eslint 对js进行检测， 修改成符合eslint格式的代码。使用框架：knockout JS, 使用google map API 完成地图显示和地图控件功能、 使用ajax对高德地图API进行请求获取地点的详细信息。
地图api提供的是http协议，gitpage是https协议，协议不同，所以不放到gitpage上了

健康记录应用

使用 gulp 工具构建，使用框架 backbone JS， 使用ajax请求Nutritionix API 查询对应的食物卡路里。 查询成功将返回该类食物的不同卡路里数， 选择食物后，将把食物添加到卡路里计算列表中，同时使用localStorage保存食物数据。
